# NoSQL БД Cassandra

* Модуль taco-cloud-data-cassandra является примером альтернативного использования NoSQL колоночной БД вместе taco-cloud-data (mongodb)

## Ключевые отличия:
* Вместо аннотации @Table (JPA) используется аннотация @Table
* Вместо аннотации @Id (JPA) используется аннотация @PrimaryKey
* Технически много отличий не нашёл, в репозиториях нужно сменить тип данных на UUID и в обще-то всё.
