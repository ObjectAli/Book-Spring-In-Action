# Spring Security

Пользователей приложения можно хранить 3 способами:
* хранилище учетных записей в памяти (InMemoryUserDetailsManager)
* хранилище учетных записей JDBC
* хранилище учетных записей LDAP
* либо своё решение

Разграничения прав доступа можно реализовать также на языке SpEL, что даёт существенные возможности гибкости.
